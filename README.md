# Lua Script?

- 1993년에 브라질에서 개발된 언어이다.
  - 최신 버전은 5.4.4로 2022년 1월 26일에 출시됐다.
- 확장 언어와 스크립트 언어를 지향하고 있다.
- C언어에서의 다음과 같은 부족한 부분에 집중했다.
  - 추상화 유지
  - 동적 자료구조
  - 중복 방지
  - 테스트와 디버깅
